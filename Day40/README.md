## 📚 배운 내용
### 피보나치 수열 (DP)
DP (Dynamic Programming)
- 중복되는 계산을 피하기 위해 배열에 결과를 저장
- 시간복잡도 O(n)

### LCS (Longest Common Subsequence)
두 문자열에서 순서를 지키며 가장 길게 공통으로 겹치는 부분 찾는 알고리즘   
시간복잡도: O(n × m)
