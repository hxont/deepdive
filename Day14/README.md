## 📚 배운 내용
### 📦 자바스크립트 객체(Object)란?
- 하나의 변수에 여러 속성(데이터)을 묶어서 저장할 수 있는 구조
```javascript
let user = {
  name: "훈티",
  age: 20,
  address: "런던"
};
```

### 🧱 자바스크립트 자료구조 – 스택(Stack) & 큐(Queue)
#### 📌 스택 (Stack)
- **후입선출 (LIFO: Last In First Out)** 구조
- 나중에 들어온 데이터가 가장 먼저 나감
- 대표 메서드: `push()`(넣기), `pop()`(빼기)
#### 📌 큐 (Queue)
- **선입선출 (FIFO: First In First Out)** 구조
- 먼저 들어온 데이터가 먼저 나감
- 대표 메서드: `push()`(넣기), `shift()`(앞에서 빼기)
