## 📚 배운 내용
### AWS S3 (Simple Storage Service)
- 이미지, PDF, 로그 등 파일 저장 및 공유용 서비스
- 버킷(Bucket): 데이터를 담는 컨테이너
- 객체(Object): 버킷 안에 들어있는 파일
- 장점:
    - 무제한 저장
    - HTTPS로 보안 강화
    - 장애 발생 시 자동 복구
